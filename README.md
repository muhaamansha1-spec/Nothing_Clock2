## ✨ Features

* **Monochrome Nothing UI:** Authentic dot-matrix clock font, raw grids, and a clean black-and-white interface.
* **AI Contextual Greetings:** Uses Gemini to generate glanceable, witty, or calm text based on the time of day, weather, or your schedule.
* **Smart Minimalist Alarms:** AI-generated subtle wake-up prompts rather than jarring, aggressive ringtones.
* **Fluid Animations:** Smooth transitions that respect the lightweight nature of Nothing OS.

## 🛠️ Tech Stack

* **Frontend:** Kotlin
* **AI Engine:** Google AI Studio (`gemini-3.5-flash`)
* **Styling:** Custom Dot-Matrix Fonts & Monochrome CSS Variables
